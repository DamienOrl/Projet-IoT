# Embedded application template

Usage:
* Clone to `your-project-dir` and enter it
* Edit readme file
* Modify remote URL with:
    ```sh
    $ git remote set-url origin git@gitlab.com:catie_sixtron/your-project-name.git'
    $ git push -u origin master
    ```
* Initialize with:

    ```sh
    $ mbed deploy
    ```
